# Task-2_data_insertion_null_handling
Task 2 of 30 day sql developer internship.

## Overview
This task focuses on populating the Library database with realistic data using DML commands — INSERT, UPDATE, DELETE — while practicing NULL handling and default behaviors.

## Key Operations Performed
- Added categories, authors, books, book_authors, loans and members.
- Handled missing fields using NULL or default values.
- Updated specific rows using WHERE.
- Deleted unused or orphaned records.
- Inserted a record using SELECT with WHERE NOT EXISTS.

## Concepts Practiced
- DML: Data Manipulation Language
- NULL vs 0
- DEFAULT and IS NULL
- Partial and selective inserts
- Updating multiple rows

*This project is part of the 30 day SQL Developer Internship – Task 2.*
